# ISMS Portal — Ready to Deploy

## Deploy to Vercel (Recommended)

1. Go to https://vercel.com and sign up free
2. Click "Add New Project"
3. Choose "Import from ZIP" or connect GitHub
4. Upload this entire folder as a ZIP
5. Vercel auto-detects Vite + React — click Deploy
6. Your portal is live in ~30 seconds

## Deploy to Netlify

1. Run locally first:
   npm install
   npm run build

2. Drag the dist/ folder to https://app.netlify.com

## Run Locally

npm install
npm run dev
→ Opens at http://localhost:5173
